# 学习通AI自动答题 — 使用教程

> 本教程面向零基础用户，每一步都有详细说明，跟着做就行。

## 这个东西是什么？

这是一个**浏览器插件的辅助脚本**。安装后，打开学习通的作业或考试页面，它会自动帮你做题：

- 自动读取页面上的题目
- 把题目发给 AI（DeepSeek 或 MiMo）
- 自动把 AI 给的答案填到页面上

支持单选、多选、判断、填空、简答等所有常见题型。

---

## 安装前的准备

你需要准备两样东西：

| 序号 | 准备什么 | 在哪里弄 |
|:---:|---|---|
| 1 | **Tampermonkey 浏览器插件** | 浏览器扩展商店（下面会教） |
| 2 | **AI 的 API Key**（一串密钥） | DeepSeek 或 MiMo 官网注册（下面会教） |

> **什么是 API Key？** 可以理解为 AI 服务的"账号密码"，脚本需要它才能调用 AI 来答题。

---

## 第一步：安装 Tampermonkey 插件

Tampermonkey（俗称"油猴"）是一个浏览器插件，用来运行各种辅助脚本。

### 用什么浏览器？

推荐使用以下任一浏览器（电脑端）：

- **Edge**（Windows 自带，推荐）
- **Chrome**（谷歌浏览器）
- **Firefox**（火狐浏览器）

### 安装步骤

1. 点击下面对应你浏览器的链接，进入插件安装页面：

   - [Edge 浏览器安装 Tampermonkey](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd)
   - [Chrome 浏览器安装 Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   - [Firefox 浏览器安装 Tampermonkey](https://addons.mozilla.org/firefox/addon/tampermonkey/)

2. 在打开的页面中，点击 **"获取"** 或 **"添加到浏览器"** 按钮

3. 浏览器会弹出提示问你是否允许，点击 **"添加扩展"** 或 **"确认"**

4. 安装完成后，浏览器右上角工具栏会出现 Tampermonkey 的图标（一个黑色圆脸上带两个角）

> **找不到图标？** 点击浏览器右上角的拼图图标 🧩（扩展管理），在列表里找到 Tampermonkey，点击旁边的图钉 📌 把它固定到工具栏。

---

## 第二步：获取 API Key

API Key 是调用 AI 答题的凭证，**二选一**即可。

### 方案 A：使用 DeepSeek（推荐）

1. 用浏览器打开 DeepSeek 官网：**https://platform.deepseek.com/**

2. 点击页面上的 **"注册"** 或 **"登录"**，用手机号注册一个账号

3. 登录后，在页面左侧菜单找到 **"API Keys"**，点击进入

4. 点击 **"创建 API Key"** 按钮

5. 给 Key 起个名字（随便写，比如"答题"），然后点确认

6. 页面会显示一串以 `sk-` 开头的密钥，**立刻复制保存下来**（比如发到微信文件传输助手），因为页面关闭后就看不到了

> **注意：** DeepSeek 的 API 可能需要充值才能使用，注册后按页面提示充值少量金额即可（一般几块钱够用很久）。

### 方案 B：使用 MiMo

1. 用浏览器打开 MiMo 官网：**https://platform.xiaomimimo.com/console**

2. 注册并登录账号

3. 在控制台中找到 **"API Keys"**，创建一个新的 Key

4. 复制保存好以 `sk-` 开头的密钥

---

## 第三步：安装答题脚本

1. 在你的电脑上找到下载好的脚本文件：**`学习通AI答题.user.js`**

   > 如果你是从别人那里收到的，确保文件后缀是 `.user.js`，文件名不要改。

2. 点击浏览器右上角的 **Tampermonkey 图标**（那个黑色小脸）

3. 在弹出的菜单中，点击 **"管理面板"**（或"管理 Userscript"）

4. 你会看到一个新页面，顶部有几个标签页，点击最右边的 **"实用工具"**

5. 在 **"从文件导入"** 这一栏，有一个虚线框区域

6. 把 `学习通AI答题.user.js` 文件**直接拖进那个虚线框里**

7. 页面会弹出确认信息，点击 **"导入"** 或 **"安装"**

8. 看到脚本列表里出现了 **"学习通AI自动答题"**，说明安装成功了

> **也可以用另一种方式：** 在管理面板中点击顶部的 **"+"** 号，会打开一个代码编辑器，把 `.user.js` 文件里的所有内容复制粘贴进去，然后按 `Ctrl + S` 保存。

---

## 第四步：配置 API Key

1. 打开学习通的任意一个页面（网址是 `chaoxing.com` 或者你的学校域名如 `xxx.edu.cn`）

2. 等页面加载完成后，**页面右下角**会出现一个浮窗，标题是 **🤖 AI 答题助手**

   > 如果没看到浮窗，检查一下 Tampermonkey 是否已启用脚本（点击油猴图标，看"学习通AI自动答题"是否打勾）

3. 在浮窗中点击 **"设置"** 标签页

4. **"AI 提供商"** 这一项，选择你注册的那个（DeepSeek 或 MiMo）

5. 在对应的 **API Key 输入框**中，粘贴你之前保存的那串 `sk-...` 密钥

6. 其他设置先不用改，默认就好

7. 点击底部的 **"保存设置"** 按钮

> **提示：** 设置只需要配置一次，以后再打开会自动记住。

---

## 第五步：开始答题

### 打开一份试卷

在学习通上打开你要做的作业或考试页面，等页面完全加载好。

### 检查题目是否被识别

1. 看右下角的浮窗，切换到 **"控制"** 标签页
2. 页面加载几秒后，浮窗会自动提示检测到了几道题
3. 你也可以点击 **"预览题目"** 按钮，切换到 **"日志"** 标签页查看识别到的题目列表

> **如果提示"未找到题目"**：可能是页面还没加载完，等几秒再试。也可以点击 **"🔍 诊断页面结构"** 按钮排查问题。

### 开始自动答题

1. 点击浮窗中的 **"开始答题"** 按钮
2. 脚本会开始逐题调用 AI 并自动填入答案
3. 切换到 **"日志"** 标签页可以实时看到答题进度
4. 顶部的进度条会显示当前完成了多少题
5. 全部答完后会提示完成

> **答题过程中想停下来？** 随时点击 **"停止"** 按钮即可中断。

### 从第几题开始

如果前面几题你已经自己做过了，可以在 **"从第几题开始"** 的输入框里填上题号（比如填 `5` 表示从第 5 题开始做），这样就不会重复做已经完成的题目。

## 常见问题

### 浮窗没有出现

- 确认你打开的是学习通的页面（网址包含 `chaoxing.com` 或 `edu.cn`）
- 点击浏览器右上角的 Tampermonkey 图标，确认 **"学习通AI自动答题"** 前面有打勾
- 如果没有打勾，点击它启用脚本，然后刷新页面

### 提示"请先配置 API Key"

- 点击浮窗的 **"设置"** 标签页
- 确认你填了 API Key 并点了 **"保存设置"**
- 确认你选的 AI 提供商和填的 Key 是对应的（DeepSeek 的 Key 选 DeepSeek，MiMo 的 Key 选 MiMo）

### AI 回答很慢或者报错

- 检查你的网络是否正常
- 确认 API Key 是否有效（可能过期或余额不足）
- DeepSeek 官网登录后可以查看余额和调用记录
- 如果网络不好，可以在设置中把 **"流式输出"** 关掉试试

### 答案没有填进去 / 填入位置不对

- 学习通不同页面的编辑器类型不一样，脚本会自动尝试多种方式填入
- 如果遇到问题，查看 **"日志"** 标签页的报错信息
- 可以点击 **"🔍 诊断页面结构"** 按钮检查页面情况

### 想用第三方 API 代理

如果你有兼容 OpenAI 格式的第三方代理地址，可以在设置的 **"自定义 Endpoint"** 中填入。留空则使用官方地址。

## 设置项速查

打开浮窗 → **"设置"** 标签页：

| 设置项 | 默认值 | 说明 |
|---|---|---|
| AI 提供商 | DeepSeek | 选你注册了的那个 |
| DeepSeek API Key | 空 | 填 `sk-` 开头的密钥 |
| MiMo API Key | 空 | 填 `sk-` 开头的密钥 |
| 自定义 Endpoint | 空 | 一般不用填，有代理才需要 |
| DeepSeek 模型 | `deepseek-v4-pro` | 不用改 |
| MiMo 模型 | `mimo-v2.5-pro` | 不用改 |
| 答题间隔 | 2000 | 两题之间的等待时间（毫秒），不用改 |
| 自动提交 | 关闭 | 想答完自动交卷就打开 |
| 流式输出 | 开启 | 不用改 |
